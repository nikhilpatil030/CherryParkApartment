


# Cherry Park Apartment Project

This repository contains the codebase for the Cherry Park Apartment project, a web application designed to showcase the features and amenities of the Cherry Park Apartments.

## Getting Started

To get started with this repository, follow these steps:

* Clone the repository using `git clone https://github.com/[Your Username]/CherryParkApartment.git`
* Install dependencies using `npm install` or `yarn install`
* Run the application using `npm start` or `yarn start`

## Features

This repository includes the following features:

* **Amenities**: Displays a list of apartment features and community amenities, including a fitness center, pool, and clubhouse.
* **Schedule a Tour**: Allows users to schedule a tour of the apartment complex.
* **Photo Gallery**: Displays a gallery of photos showcasing the apartment complex and its amenities.
* **Floor Plans**: Displays a list of available floor plans, including studio, one-bedroom, and two-bedroom options.
* **Neighborhood**: Provides information about the surrounding neighborhood, including local restaurants, shops, and attractions.

## Code Structure

The code is organized into the following directories:

* `src/app/navbar`: Contains the components for the navigation bar.
* `src/app/home`: Contains the components for the home page, including the amenities, schedule a tour, and photo gallery features.
* `src/app/floor-plans`: Contains the components for the floor plans feature.
* `src/app/neighborhood`: Contains the components for the neighborhood feature.
* `src/assets`: Contains images and other assets used throughout the application.
